Unit Testing React Applications with Mocha, Chai and Enzyme Talk

Posted on Jun 24, 2016
#react #testing #mocha #chai #enzyme #talks

In April 2016, I gave a talk at Red Badger’s React London User Group meetup in London, on the topic of “Unit Testing React Applications with Mocha, Chai and Enzyme”.

The talk introduces the fundamentals of testing a React application with AirBnB’s excellent Enzyme library, and explains my approach to both unit and integration testing apps using the combination of Mocha, Chai and Enzyme.

Both the code samples, and the slides are available on GitHub.
