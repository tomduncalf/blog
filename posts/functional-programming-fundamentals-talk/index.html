Functional Programming Fundamentals Talk

Posted on Jun 24, 2016
#functional programming #talks

In February 2016, I gave a talk at Red Badger’s Functional Programming for the Web meetup in London, on the topic of “Functional Programming Fundamentals”.

The talk explains topics such as first class functions, composition, currying, functors and monads in a (hopefully!) simple way for newcomers to FP, illustrated using JavaScript examples which are available on GitHub, as are the slides.

The talk was inspired by Dr. Boolean’s excellent free e-book, Professor Frisby’s Mostly Adequate Guide to Functional Programming.
